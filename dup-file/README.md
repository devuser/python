# 复制文件内容n份

在日常工作中经常遇到如下连续动作

1. 将源文件内容复制n份，忽略其中的结束标志行
2. 添加结束标志行
3. 写入目标文件

针对上述任务，我写了这个Python程序，鼓励使用。

缺省复制当前目录下`src.P`文件到目标文件`dst.P`。

复制源文件中的内容，注意每行文本必须包含或超过4个字符，且忽略结束标志行。

结束标志行，由`TLRL`引导加15位数字的有效行数（不含结束标志行），不足15位前补零。

## 使用方法

```
python main.py
cat dst.P
```

注意使用前修改`main.py`中的`src.P`和`dst.P`的文件路径。
